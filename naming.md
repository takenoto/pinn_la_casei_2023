
# Naming convention

Loss Teste/Validation => LoV
Loss Train => LoT

Learning Rate => LR


- Reatores:
  - R => Reação sem nada
  - B => Batelada
  - CSTR => É de fato um CSTR, já inicia no volume máximo e só vamos ver as concentrações variando.
  - CR1 => CR que possui Vo = 1L
  - CR2 => CR que possui Vo = 0.1L
